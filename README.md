# Ojo

an open soure gtk media player for the MATE desktop

![Ojo](/data/OjoScreenshot.png)

## compile

```bash
git clone https://github.com/FreaxMATE/Ojo.git
cd Ojo/src/
make
./ojo
```


## install (experimental)

```bash
make
sudo make install
```
## uninstall (experimental)

```bash
make
sudo make uninstall
```
