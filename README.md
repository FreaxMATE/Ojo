# Ojo

an open soure gtk media player for the MATE desktop

![Ojo](/data/OjoScreenshot.png)

## compile

```bash
git clone https://github.com/FreaxMATE/Ojo.git
cd Ojo/
make
./ojo
```


## install (experimental, not working properly yet)

