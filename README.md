# Ojo

Ojo is an open soure gtk media player for linux.

![Ojo](/data/OjoScreenshot.png)

## Installation
### Install from source
```bash
git clone https://github.com/FreaxMATE/Ojo.git
cd Ojo/
meson build && cd build
ninja
sudo ninja install
```
